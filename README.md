# Word Counter

A simple command-line program in C that counts the total number of words in a text file and optionally counts occurrences of a specific word.

## Compilation

```bash
gcc word_counter.c main.c -o word_counter.exe
```
## Usage 
```bash
./word_counter.exe
```
1. Enter the path to your .txt file.
2. Optionally, enter a word to track.
3. The program outputs the total word count and occurrences of the chosen word.
